# VIBE NET - HTML Version

Jednoduché HTML řešení pro web VIBE NET.

## ✅ Co obsahuje:

- `public/index.html` - Kompletní design s Tailwind CSS
- Všechny animace a efekty
- Responsive design

## 🚀 Jak nasadit:

1. Pushujete na GitHub
2. Connectnete s Vercel
3. Vercel automaticky nasadí!

## ✏️ Jak editovat:

Otevřete `public/index.html` a editujte text přímo!

Když uložíte a pushujete na GitHub, web se automaticky aktualizuje! 🎉
